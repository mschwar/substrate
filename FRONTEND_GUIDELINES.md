
# Frontend / UI Guidelines

**Platforms**: Desktop only (macOS, Windows, Linux)

**Core Views (MVP)**

- Inbox
- Search
- Item view
- Timeline
- Dedupe review
- Import jobs
- Settings

**Design Principles**

- Progressive disclosure (beginner â†’ advanced)
- Keyboard-first (shortcuts mandatory)
- Accessibility: WCAG AA, ARIA, keyboard nav, no color-only info
- Themes: light/dark system follow

**Item View Mandatory Elements**

- Title + badges
- Editable frontmatter (live validation)
- Provenance drawer (observed claims, confidence bars)
- Attachments list
- Backlinks
- Cite button

**Interaction Rules**

- No auto-applied changes (privacy, tags)
- All destructive ops require confirmation phrase
- Warnings for low confidence, conflicts, large files

**Keyboard Defaults** (customizable)

- / search
- n new note
- i inbox
- t timeline
- cmd/ctrl+k palette
